The verben-visible-column component is a customizable interface that allows users to show or hide table columns based on their selections. It accepts various input properties, including border, borderRadius, boxShadow, bgColor, width, textColor, pd, and color options for primary and secondary elements. The component emits an event through the (columnsUpdated) output whenever the visibility of columns is changed, enabling dynamic updates. This component provides a flexible styling system to cater to a variety of design needs.
